MySQL

Oracle MySQL Database Service는
전 세계 개발자들이 가장 많이 사용하는
오픈 소스 데이터베이스인
MySQL을 기반으로,
안전한
클라우드 네이티브 애플리케이션을
빠르게 개발하고 배포할 수 있도록
지원하는
완전 관리형
데이터베이스 서비스입니다.

MySQL Database Service는 통합된 고성능 인메모리 쿼리 가속기인 HeatWave를 갖춘
유일한 MySQL Database에 대해 직접 정교한 분석을 실행 할 수 있으므로,
복잡하고 시간이 많이 걸리며 비용이 많이 드는 데이터 이동 및
별도의 분석 데이터베이스와의 통합이 필요하지 않습니다.

HeatWave는 분석 및 혼합 워크로드에서 MySQL 성능을 수십배 가속화합니다.
오라클 클라우드 인프라스트럭쳐(OCI)에 최적화된 MySQL Database Service는
OCI 및 MySQL엔지니어링 팀에서 100%구축, 관리 및 지원합니다.
