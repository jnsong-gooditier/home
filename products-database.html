Database

Oracle Database는
on-premises와 cloud 모두에서
업계 최고의 성능, 확장성,
안정성 및 보안을 제공합니다.

Oracle Database 19c는 현재 long term 릴리스이며
최고 수준의 릴리스 안정성과 지원 및
버그 수정을 위한 최장 기간을 제공합니다.
현재 innovation 릴리스로 production 용도로도 제공되는 Oracle Database 21c는 많은 개선 사항과
새로운 기능에 대한 초기 통찰력을 제공합니다.

여기에는 자율 관리, 데이터베이스 내 Javascript 및
기본 블록체인 테이블을 통한 향상된 다중 모델 지원,
AutoML 및 향후 long term 릴리스에 통합될
샤딩 향상과 같은 다중 직업 부하 개선이 포함됩니다.

▾ Oralce Database 주요기능

Oracle Database In-Memory 및 Oracle Multitenant와 같은 고급 데이터 data warehousing 및 분석 기술을 통해 분석 팀은
확장 가능한 데이터 data warehouses에 대한
보다 심층적인 분석을 더 짧은 시간에
완료할 수 있습니다. 고객은 on-premises 또는 Oracle Cloud Infrastructure에서
Oracle Database 기술을 사용하여
보다 심층적인 데이터 기반 통찰력을 개발합니다.

관련 기능 및 tool

  • - Oracle Partitioning
  • - Oracle Spatial
  • - Oracle Multitenant
  • - Oracle Database In-Memory
  • - Oracle Graph
  • - Oracle Big Data Connectors

개발자는 SQL, JSON, XML 및 다양한 절차적 언어를 사용하여 확장 가능한 고성능 애플리케이션을 빠르게 만들 수 있습니다.
Oracle Database는 APEX 및 통합 데이터베이스 기능과 같은 다양한 기본 제공 개발 도구를 제공합니다.

관련 기능 및 tool

  • - Oracle Application Express (APEX)
  • - Machine Learning
  • - Spatial and Graph features
  • - Java
  • - Node.js
  • - Oracle SQL Developer
  • - Oracle JSON Document Database
  • - SQL
  • - Python
  • - .Net

Oracle Database에는 Oracle Active Data Guard 및 Oracle Real Application Clusters를 비롯한 내장 기능과 옵션이 포함되어 있어 고객 데이터베이스를 효율적으로 확장하고 통합할 수 있습니다.

관련 기능 및 tool

  • - Oracle Real Application Clusters (RAC)
  • - Active Data Guard
  • - Oracle Multitenant
  • - Oracle Site Guard
  • - Oracle’s Zero Data Loss Recovery
  • - Oracle RAC One Node
  • - Oracle GoldenGate
  • - Oracle Sharding
  • - Oracle Exadata

Oracle Database는 데이터베이스 내부에서 실행되는 강력한 알고리즘으로 머신 러닝(ML)을 가속화하므로 고객은 데이터를 이동하거나 다시 포맷할 필요 없이 ML 모델을 구축하고 실행할 수 있습니다. 데이터 과학자는 Python, R, SQL 및 기타 도구를 활용하여 ML 기능을 데이터베이스 애플리케이션에 통합하고 사용하기 쉬운 대시보드에서 분석 결과를 제공합니다.

관련 기능 및 tool

  • - Oracle Data Miner
  • - Oracle Machine Learning for SQL
  • - Oracle Machine Learning for Spark
  • - Oracle Site Guard
  • - Oracle Machine learning notebooks
  • - Oracle Machine Learning for R
  • - Oracle Analytics Cloud

단일 창 관리 대시보드를 통한 일관된 관리 프로세스로 전사적 데이터베이스 성능과 가용성을 높입니다. DBA는 Oracle 데이터베이스 관리 솔루션을 사용하여 on premises, Oracle Cloud Infrastructure 및 타사 cloud에서 실행되는 데이터베이스의 모니터링 및 관리를 통합하여 워크로드를 줄입니다.

관련 기능 및 tool

  • - Enterprise Manager Cloud Control

최적화된 짧은 대기 시간 및 높은 처리량을 위해 설계된 다양한 기능을 사용하여 실시간 환경 및 데이터 센터 배포에서 엄격한 성능 요구 사항을 충족합니다.

관련 기능 및 tool

  • - Database In Memory
  • - Advanced Compression
  • - Real Application Testing
  • - Times Ten
  • - Partitioning
  • - Exadata

Oracle의 데이터베이스 보안 솔루션은 데이터 침해 위험을 줄이고 고객이 규정 준수 요구 사항을 보다 쉽게 충족할 수 있도록 합니다. 암호화, 데이터 마스킹, 권한 있는 사용자 액세스 제어, 활동 모니터링 및 감사를 통해 Oracle Database 환경을 강력하게 보호하고 잠재적인 취약성을 이해할 수 있습니다.

관련 기능 및 tool

  • - Oracle Advanced Security
  • - Oracle Data Masking and Subsetting
  • - Oracle Database Security Assessment Tool
  • - Oracle Key Vault
  • - Oracle Audit Vault and Database Firewall
  • - Oracle Data Safe
  • - Oracle Database Vault
  • - Oracle Label Security

Oracle Cloud Infrastructure의 최적화된 데이터베이스 기능 제품군을 통해 고객은 특정 요구 사항에 맞게 리소스를 긴밀하게 조정하여 효율적이고 저렴한 비용으로 운영할 수 있습니다. 기업은 완전 관리형 Autonomous Database 기능, Exadata Cloud 서비스의 성능 및 제어, Oracle Database, MySQL 데이터베이스 및 NoSQL 데이터베이스 클라우드 서비스의 저렴한 비용을 결합하여 디지털 혁신을 구현합니다.

관련 기능 및 tool

  • - Autonomous Data Warehouse
  • - Autonomous JSON Database
  • - Oracle Exadata Cloud@Customer solutions
  • - Oracle Database Cloud Services on Bare Metal
  • - Oracle NoSQL Database Cloud Service
  • - Autonomous Transaction Processing
  • - Oracle Exadata Cloud Service
  • - Oracle Database Cloud Services on Virtual Machines
  • - Oracle MySQL Database Service
  • - Oracle APEX Application Development
